==================
Running bilby_pipe
==================

To run :code:`bilby_pipe`, one first needs to define an `ini file <ini_file.txt>`_.
As an example, here is the example ini file for GW150915:

.. literalinclude:: ../example_ini_files/GW150914.ini

Then to run and submit a job,

.. code-block:: console

   $ bilby_pipe GW150914.ini --submit
